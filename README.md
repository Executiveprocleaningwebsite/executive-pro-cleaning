## Quick start
1) Open this folder in **VS Code**
2) Install dependencies:
   ```bash
   npm install
   ```
3) Create your env file:
   - Copy `.env.example` to `.env.local`
   - Fill in the Turnstile keys + Gmail App Password

4) Start the dev server:
   ```bash
   npm run dev
   ```
5) Visit: http://localhost:3000

## Notes
- Pages live in `/app`
- Shared nav/footer live in `/components`
- Images live in `/public/images`
- Contact form backend lives in `/app/api/contact/route.js`
